# 🧩 **Question 1**

## **1. GET /api/students**

Retrieve a list of all students, supporting both JSON and XML formats, and calculate their current GPA.

### Requirements:

- Return a list of students with:
  - StudentId
  - StudentName
  - Email
  - GPA

- GPA is calculated as:
  - Average of all Grade values in the Enrollments table
  - Exclude records with NULL Grade

- Support response formats:
  - JSON
  - XML (using media formatters)

### Example:

```
GET http://localhost:5000/api/students
```

---

## **2. GET /api/student-performance**

Retrieve a list of students similar to `/api/students` but with filtering and pagination.

### Filtering:

- `minGpa`: Only return students with GPA ≥ value
- `studentName`: Search by name (contains)

### Pagination:

- `page`: page number
- `pageSize`: number of students per page (default = 10)

### Response must include:

- data (list of students)
- totalStudents
- totalPages
- currentPage
- pageSize

### Error handling:

- If `page` or `pageSize` ≤ 0 → return **400 Bad Request**

### Example:

```
GET http://localhost:5000/api/student-performance?minGpa=8&page=-1&pageSize=18
→ 400 Bad Request
→ "Invalid pagination parameters"
```

```
GET http://localhost:5000/api/student-performance?minGpa=8&page=1&pageSize=10
→ 200 OK
```

---

## **3. PUT /api/enrollments/{enrollmentId}/grade**

Update the grade for an enrollment.

### Requirements:

- Input: Grade (float)

### Validation:

- Grade must be in range **[0, 10]**
  - Otherwise → **400 Bad Request**
  - Message: `"Grade must be between 0 and 10"`

- If enrollmentId does not exist → **404 Not Found**

### Response:

- enrollmentId
- studentId
- grade

### Example:

```
PUT http://localhost:5000/api/enrollments/8/grade
Body:
{
  "grade": 5
}
```

---

## **4. DELETE /api/enrollments/{enrollmentId}**

Cancel a student's enrollment.

### Requirements:

- Delete record from Enrollments table
- If success → return **204 No Content**

### Error handling:

- If Grade is NOT NULL:
  - → **400 Bad Request**
  - Message: `"Cannot cancel an enrollment that has already been graded"`

- If enrollment not found:
  - → **404 Not Found**
  - Message: `"No enrollment found with provided EnrollmentId"`

### Example:

```
DELETE http://localhost:5000/api/enrollments/9
→ 400 Bad Request
→ Cannot cancel an enrollment that has already been graded
```
